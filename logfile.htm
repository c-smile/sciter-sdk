  • [script] Breaking change: non-safe script features are disabled by default. To enable them use this explicit declaration: 
    SciterSetOption(NULL, SCITER_SET_SCRIPT_RUNTIME_FEATURES,  
   ALLOW_FILE_IO | 
   ALLOW_SOCKET_IO | 
   ALLOW_EVAL | 
   ALLOW_SYSINFO );
